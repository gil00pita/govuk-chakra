# Typography

[Text](https://www.chakra-ui.com/docs/components/text) component is used to render text and paragraphs within an interface.

The [Heading](https://www.chakra-ui.com/docs/components/heading) component is used to render headings and titles.

[GOV.UK Design System Typography guidelines](https://design-system.service.gov.uk/styles/typeface/).

## Type Scale

GOV.UK Type Scale: **16**, **19**, **24**, **27**, **36**, **48**, **80**.

### Responsive behaviour

The type scale changes based on screen size.

The scale for **large screens** is used when the screen is wider than the tablet breakpoint (640px).

The component will auto adjust the font size depending of the screen size.
